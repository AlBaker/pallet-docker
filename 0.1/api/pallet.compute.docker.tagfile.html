pallet.compute.docker.tagfile documentation

Manages a file on a docker host, containing tags for docker containers.

The tag file is a clojure datastructure, that is a map from node id to a map from tag keywords to sting value.

The tags are kept in the settings for the docker host node.

Locking is taken care of at the node level, so there can be multiple docker compute services talking to a single docker host.
